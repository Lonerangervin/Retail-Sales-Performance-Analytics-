Retail Sales Power BI Dashboard
Overview

This repository contains a **Retail Sales Analysis** project built using **Power BI** and a CSV dataset. The project focuses on analyzing retail store sales performance through interactive dashboards, highlighting trends, patterns, and key business insights that support data-driven decision-making.

The repository includes:

* A Power BI visualization file (`.pbix`)
* The raw retail sales dataset (`.csv`) used for analysis

Objectives

* Analyze overall retail sales performance
* Identify sales trends across time
* Compare performance by product category and other dimensions
* Demonstrate data cleaning, modeling, and visualization in Power BI

Tools & Technologies

Power BI – Data modeling, visualization, and dashboard creation
CSV Dataset – Retail store sales data
Data Analysis Concepts – Aggregation, trends, KPIs, filtering
SQL for analysis purposes 


Dataset

The dataset (`retail_store_sales.csv`) contains retail transaction-level data used to generate insights in the dashboard.

Typical fields include (may vary depending on version):

* Date
* Product / Category
* Sales Amount
* Quantity Sold
* Store or Region information

 Dashboard Features

The Power BI dashboard provides:

* Total Sales Overview – High-level KPIs summarizing performance
* Sales Trends – Time-based analysis (monthly/yearly)
* Category Performance – Comparison of sales across product categories
* Interactive Filters – Dynamic exploration of data
